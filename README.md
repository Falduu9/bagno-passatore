# 🏖️ Bagno del Passatore - Sito Web Ufficiale

Sito web responsive sviluppato per il **Bagno del Passatore**, stabilimento balneare situato a Lido di Dante (Ravenna).

 **Sito live:** [bagnodelpassatore.it](https://bagnodelpassatore.it)

## 📋 Descrizione

Sito vetrina one-page per un'attività balneare che offre:
- Ristorante e Pizzeria
- Noleggio ombrelloni e lettini
- Campi da Beach Tennis e Padel
- Area Beach Dog

Il sito è stato progettato per essere **completamente statico** e a **manutenzione zero**, permettendo al cliente di avere una presenza online professionale senza costi ricorrenti di gestione.

## 🛠️ Tecnologie Utilizzate

- **HTML5** - Struttura semantica
- **CSS3** - Styling con Flexbox e Grid, design responsive
- **JavaScript (Vanilla)** - Interattività (navbar sticky, menu mobile, smooth scroll)
- **GitHub Pages** - Hosting e deployment
- **Register.it** - Gestione dominio

## ✨ Funzionalità

- ✅ Design completamente responsive (mobile-first)
- ✅ Navbar sticky con effetto glassmorphism allo scroll
- ✅ Menu hamburger animato per mobile
- ✅ Smooth scroll tra sezioni
- ✅ Cookie banner conforme GDPR
- ✅ Privacy Policy
- ✅ Link diretto chiamata (`tel:`) e Google Maps
- ✅ Integrazione menu PDF
